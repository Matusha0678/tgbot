# Holiday Notification Telegram Bot

Телеграм бот, который предупреждает о праздниках за один день до них.

## Локальный запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python bot_simple.py
```

## Развертывание на Railway

### Шаг 1: Подготовка репозитория

1. Создайте GitHub репозиторий с файлами:
   - `bot_simple.py` - основной файл бота
   - `requirements.txt` - зависимости
   - `railway.toml` - конфигурация Railway
   - `Procfile` - команда запуска

### Шаг 2: Настройка Railway

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Подключите свой GitHub аккаунт
3. Нажмите "New Project" → "Deploy from GitHub repo"
4. Выберите репозиторий с ботом

## Команды

- `/start` - Приветствие и информация о боте
- `/addme` - Добавить себя в список уведомлений
- `/removeme` - Удалить себя из списка уведомлений  
- `/holidays` - Показать все праздники

## Как это работает

Бот проверяет праздники каждый час и отправляет уведомления за 24 часа до наступления праздника всем пользователям, которые добавили себя в список уведомлений командой `/addme`.
