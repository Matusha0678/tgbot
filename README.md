# Holiday Notification Telegram Bot

Телеграм бот, который предупреждает о праздниках за один день до них.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота у [@BotFather](https://t.me/botfather)

3. Замените `YOUR_BOT_TOKEN` в файле `bot.py` на ваш токен

4. Запустите бота:
```bash
python bot.py
```

## Команды

- `/start` - Приветствие и информация о боте
- `/addme` - Добавить себя в список уведомлений
- `/removeme` - Удалить себя из списка уведомлений  
- `/holidays` - Показать все праздники

## Как это работает

Бот проверяет праздники каждый день и отправляет уведомления за 24 часа до наступления праздника всем пользователям, которые добавили себя в список уведомлений командой `/addme`.
